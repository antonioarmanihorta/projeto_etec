# projeto_etec

Repositorio do projeto da ETEC com a aplicacao organizada entre frontend e backend.

## Estrutura

```text
projeto_etec/
|-- README.md
|-- requirements.txt
`-- app/
    |-- __init__.py
    |-- main.py
    `-- pages/
        |-- __init__.py
        |-- historico_jogador.py
        |-- login.py
        |-- modos_jogos.py
        |-- questoes.py
        |-- teacher.py
        |-- classroom_code.py
        `-- add_question.py
```

## Instalacao do frontend

```bash
python -m venv .venv
```

```bash
.venv\Scripts\activate
```

```bash
pip install -r requirements.txt
```

## Executar

```bash
python -m app.main
```
